# scrapingHomework
## Description

This scraping app scrapes article titles and links from bbc.com.

Available routes:

* /
-shows front-end page

* /scrape
-performs scraping

* /articles
-shows raw scraped data

## Problem:   
Scrape article data from a news website.

## Solution: 
Use:
 - Cheerio
 - Express
 - Mongoose
 - axios
